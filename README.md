# Toho-Project-Second-games @ Picture-RPG Powered by zuyasi

東方Projectの二次創作ブラウザゲーム「Picture-RPG（仮）」のリポジトリです。  
画像とHTMLベースで進行する、ストーリー選択型のRPGを制作しています。

## リポジトリ構成

| パス | 内容 |
| --- | --- |
| `プログラム/HTML/` | ゲーム本編の画面HTML（タイトル、主人公選択、中断、シナリオ画面など） |
| `プログラム/JSON/` | 画面や進行に対応するデータJSON |
| `プログラム/template-image-editor.html` | templateフォルダ内の画像を読み込んで配置・編集するエディタ |
| `template/eyes/` | 目として追加できるテンプレート画像 |
| `template/balls/` | ボール系テンプレート画像 |
| `template/shapes/` | 図形テンプレート画像 |
| `写真/` | 背景・キャラクター・ボタンなどの画像素材 |
| `音声/` | ゲームで利用する音声素材 |
| `credit.html` | クレジットページ |

## 補足

- 画面遷移はHTML同士のリンクを中心に構成されています。
- 素材クレジットは `credit.html` を参照してください。
